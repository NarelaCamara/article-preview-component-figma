# React + TypeScript + Vite

This project is an article preview component built with React, TypeScript, and Vite. It showcases a responsive card component that displays article information with a preview feature, designed as a solution to the Frontend Mentor challenge.


### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/
├── styles/
└── App.tsx
```
